# NAO-AI Setup Guide

## Prerequisites:
1. Docker installed.
2. Python 3.8+ installed.
3. Node.js installed.

### Steps to Run:
1. Install backend dependencies:
   ```
   pip install -r requirements.txt
   ```

2. Install frontend dependencies:
   ```
   cd frontend
   npm install
   ```

3. Run the application:
   ```
   docker-compose up --build
   ```
